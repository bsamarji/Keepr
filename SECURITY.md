# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

To report a security issue, please use the GitHub Security Advisory ["Report a Vulnerability"](https://github.com/bsamarji/Keepr/security/advisories/new) tab.

One of the maintainers will respond to your issue indicating the next steps to be taken, and will keep you posted on the progress of the fix and they may ask you further questions or guidance about the security issue.

If your issue does not get a response within 5 business days, you can escalate your issue by emailing one of the maintainers directly or the project owner (bensamarji5637@gmail.com).
